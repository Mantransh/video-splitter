🎬 Video Splitter Web App

A full-stack web application that allows users to upload videos and generate short clips of customizable duration using FFmpeg.

🚀 Features
Upload video files (up to 100MB)
Split videos into multiple clips (30s, 45s, 60s, 90s)
Real-time processing UI
Clean and responsive interface

🛠 Tech Stack
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Video Processing: FFmpeg

⚙️ How It Works
User uploads a video
Backend processes it using FFmpeg
Video is split into multiple clips
URLs are returned and displayed in UI
📦 Setup Instructions
Clone repo

git clone https://github.com/Mantransh/video-splitter.git

Install dependencies

cd frontend
npm install

cd ../backend
npm install

Run project

backend
npm start

frontend
npm run dev
